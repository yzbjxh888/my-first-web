Hermes Agent 是我的核心项目，一个基于 MCP 协议的智能体系统。

它通过原生支持 Model Context Protocol，能够无缝集成各种外部工具。它不仅仅是一个聊天机器人，更是一个能够处理任务、编写代码、管理项目并进行研究的自动化专家。
