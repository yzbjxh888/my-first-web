🧑 网站主人

张俊翔 (Zhang Junxiang)，全栈开发者与 AI 基础设施工程师。 专注于构建 Agentic DevOps 系统 — 将大型语言模型与传统工程工具深度整合， 实现从代码编写、服务部署到内容生成的全面自动化。

技术栈覆盖前后端、数据库、AI/ML 全链路。 日常开发环境为 WSL (Windows Subsystem for Linux) + VS Code， GitHub 托管代码，Vercel 部署前端。

🤖 Hermes Agent

Hermes Agent 是一个开源 AI 编程助手， 具备 50+ 可加载技能、原生 MCP 客户端、多平台消息适配、 定时任务调度（Cron）、Webhook 订阅、并行子代理等强大能力。

Hermes 不仅是代码助手，更是一个完整的 Agentic 操作系统： 它能管理 systemd 服务、操作数据库、爬取网页、生成内容、 监控系统状态、控制智能家居设备，并通过 Telegram/Discord/Slack 等平台与你交互。
